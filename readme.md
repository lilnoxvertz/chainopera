## SETUP

1. ```bash
    npm install
   ```

2. fill up wallet.txt
3. fill up proxy.txt if u have one
4. fill up groq api key on .env

## FEATURES

1. support multi account
2. support proxy
3. support generating wallet
4. ai interact
5. check in
6. fetch points

## COMMANDS

- CHECK IN

```bash
    npm run cekin
```

- INTERACT WITH AI

```bash
    npm run interact
```

- GENERATING WALLET

1. go to src/main/generate.wallet.js and fill any amount

```bash
    npm run generate
```

- FETCHING POINTS

```bash
    npm run point
```
